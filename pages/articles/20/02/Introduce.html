一个一个介绍吧

info_of_str

关于字符串的配置信息可以放在 [info_of_str] 下面，同样的关于数字的配置信息可以放在 [info_of_int] 下面。

获取配置文件中的信息的程序是"getConfig.py" ``` python from configparser import SafeConfigParser

def get_config(config_file='info.ini'): parser = SafeConfigParser() parser.read(config_file) # get the ints, floats and strings #_conf_floats = [ (key, float(value)) for key,value in parser.items('floats') ] _conf_ints = [ (key, int(value)) for key,value in parser.items('info_of_int') ] _conf_strs = [ (key, str(value)) for key,value in parser.items('info_of_str') ] return dict(_conf_ints + _conf_strs)

```

以上的代码可以返回一个字典类型的数据,类似json ``` python {'num_page': 10, 'name': 'PitiKid', 'blog_name': '我的小博客', 'quote': '退却只会衰老,胆小必招死亡', 'domain': 'https://pitikid.gitee.io/', 'project_root': '/home/pitikid/thc/t_git/my_blog/', 'template_root': 'templates/', 'static_root': 'static/', 'articles_root': 'articles/'}

```
